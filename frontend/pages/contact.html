Frequently Asked Questions

Find quick answers to common questions about our service.

What file formats do you support?

We support over 50 file formats including:

  • Images: PNG, JPG, JPEG, WEBP, BMP, GIF, SVG, TIFF
  • Videos: MP4, MOV, AVI, MKV, WMV, FLV, WEBM, MPEG
  • Audio: MP3, WAV, AAC, FLAC, OGG, M4A, AIFF, WMA
  • Documents: PDF, DOC, DOCX, TXT, PPT, PPTX, XLS, XLSX, RTF
Is there a file size limit?

Yes, we have different limits based on your account type:

  • Free Account: 100MB per file
  • Pro Account: 1GB per file
  • Business Account: 5GB per file

For larger files, please contact our enterprise support.

How long are converted files stored?

File retention periods vary by account type:

  • Free Account: 24 hours
  • Pro Account: 7 days
  • Business Account: 30 days

You can always re-convert files if needed. Important files should be downloaded promptly.

Is my data secure?

Absolutely! We take security very seriously:

  • 256-bit SSL encryption for all file transfers
  • Files are automatically deleted after retention period
  • No third-party access to your files
  • GDPR and CCPA compliant
  • Regular security audits
Can I convert files on mobile devices?

Yes! Our website is fully responsive and works on all devices:

  • Mobile browsers (Chrome, Safari, Firefox)
  • Tablet devices
  • Desktop computers

We also have mobile apps available on:

  • iOS App Store
  • Google Play Store
Do you offer API access?

Yes! We offer a comprehensive API for developers:

  • RESTful API with JSON responses
  • Support for all conversion types
  • Webhook notifications
  • Detailed documentation
  • Free tier with 100 conversions/month

Visit our Developer Portal for more information.
